## 📖 About

This repository serves as my **Data Structures & Algorithms practice journal**, where I consistently solve algorithmic problems to strengthen problem-solving skills, improve coding proficiency, and prepare for technical interviews.

Every accepted solution is **automatically synchronized** from **LeetCode** using **LeetSync**, making this repository a continuously growing record of my DSA journey—without any manual uploads or maintenance.

### ✨ Highlights

- 📚 Dedicated repository for **DSA practice and interview preparation**
- ⚡ Every accepted LeetCode solution is synced automatically via **LeetSync**
- ✅ Contains only accepted and verified solutions
- 📈 Progress dashboard updates automatically as new problems are solved
- 🔄 Zero manual effort required to keep the repository up to date
